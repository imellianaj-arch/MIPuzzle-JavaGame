MI PUZZLE - Java Desktop Game
MI PUZZLE is an interactive desktop-based puzzle game developed using Java. 
The project demonstrates the implementation of graphical user interfaces (GUI), game logic, event handling, 
and object-oriented programming concepts within a desktop application environment.
This project was developed as part of an academic learning journey in Information Systems and Entrepreneurship, 
focusing on software development, problem-solving, and user interaction design.

Game Overview:
The game begins with a set of numbered tiles arranged in a random order. Players must strategically move the tiles 
and rearrange them into the correct numerical sequence to complete the puzzle. The game challenges logical thinking, 
pattern recognition, and problem-solving skills.

Technologies Used:
- Programming Language: Java
- GUI Framework: Java Swing
- Development Environment: Apache NetBeans IDE 16
- Build Tool: Maven
- Programming Paradigm: Object-Oriented Programming (OOP)

How to Download and Play:
Since this is a desktop application, you only need Java installed on your computer to run the game.
1. Navigate to the Releases section on the right side of this repository.
2. Download the latest .jar file (e.g., FinalProjectLABProgramming-1.0-SNAPSHOT.jar).
3. Ensure that Java Runtime Environment (JRE) is installed on your device.
4. Double-click the downloaded .jar file to launch and play the game.
